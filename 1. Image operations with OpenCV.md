# Image operations with OpenCV
### Importing OpenCV library
```python
import cv2
```
### Reading image
```python
img.read("path")
```
### Resizing image
```python
shape = img.shape
print(shape)
```

```python
img = cv2.cvtColor(img,cv2.COLOR_BGR2GRAY)
```
